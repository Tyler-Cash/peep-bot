# Authentication Flow Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete the authentication flow to match the design handoff — pixel-perfect login page, correct loading overlay, and middleware that redirects already-authenticated users away from `/login`.

**Architecture:** The backend owns OAuth2 (Spring Security + Discord). The frontend's job is to (a) redirect unauthenticated users to `/login`, (b) redirect the backend's OAuth endpoint on CTA click, and (c) show a loading overlay while in-flight. Middleware checks the `SESSION` cookie; already-authenticated sessions are redirected to `/` on the `/login` route.

**Tech Stack:** Next.js 15 App Router, Tailwind CSS (config already has all tokens), SWR (`useCurrentUser` → `/auth/is-logged-in`), `next/navigation` for redirects.

- [ ] **Step 3: Update middleware to add already-signed-in redirect**

  Replace the content of `frontend/src/middleware.ts` with:

  ```ts
  import { NextResponse, type NextRequest } from "next/server";

  const PUBLIC_PATHS = ["/login"];

  export function middleware(req: NextRequest) {
    const mode = process.env.NEXT_PUBLIC_API_MODE ?? "mock";
    if (mode === "mock") return NextResponse.next();

    const { pathname } = req.nextUrl;
    const hasSession = req.cookies.has("SESSION");

    // Already authenticated — bounce away from login page
    if (pathname.startsWith("/login") && hasSession) {
      const url = req.nextUrl.clone();
      url.pathname = "/";
      return NextResponse.redirect(url);
    }

    if (PUBLIC_PATHS.some((p) => pathname.startsWith(p))) return NextResponse.next();

    if (!hasSession) {
      const url = req.nextUrl.clone();
      url.pathname = "/login";
      return NextResponse.redirect(url);
    }
    return NextResponse.next();
  }

  export const config = {
    matcher: ["/((?!_next/|favicon.ico|mockServiceWorker.js).*)"],
  };
  ```
